# RedSpace

Run project locally
 ### Run the API Server
    
    In a terminal:

    ```bash
    # Initial setup
    npm install or yarn

    # Start the Server
    npm start
    ```

 ### Run the React UI

    ```bash
    # Change directory, first
    cd client/

    # Initial setup
    - npm install or yarn

    # Start the Client
    npm start
    ```
